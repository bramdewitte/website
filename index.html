About the conference

Multi-Mania is one of the biggest tech conferences of the Benelux, with 28 speakers, coming from 4 countries, giving sessions in 5 different rooms. In 2019 we will be bigger than ever because there will be some special acts, more info about this is coming soon, so be sure to keep track of us.

This edition we will have some special guest like Aral Balkan, Michael O'Neill, Nicky Lauwerijssen, ... The list is almost endless. Multi-Mania takes place in "Schouwburg" in Kortrijk, Belgium. We are easy accessible using the train, it's a 5 minute walk from Kortrijk Station.
